# .deb-for-1.7.4
Реализован запрет на установку с помощью:
```
if [ "$(grep -oP 'Astra Linux \K([0-9]+\.[0-9]+\.[0-9]+)' /etc/issue)" != "1.7.4" ]; then
        echo "Ошибка: Этот Deb-пакет предназначен для установки только на Astra Linux 1.7.4"
         exit 1
fi
```
